# üöÄ NvChad Cheat Sheet for DevOps/SysAdmin

Daftar shortcut wajib hafal untuk produktivitas maksimal di terminal.

## üìÇ File Management (NvimTree)
**Toggle Explorer:** `Spasi + e`
*(Pastikan kursor fokus di jendela Explorer)*

| Shortcut | Aksi | Keterangan |
| :--- | :--- | :--- |
| **`a`** | **Create** | Bikin file (`script.sh`) atau folder (`folder/`). |
| **`d`** | **Delete** | Hapus file/folder. |
| **`r`** | **Rename** | Ganti nama file. |
| **`y`** | **Copy Name** | Copy nama file. |
| **`c`** | **Copy** | Copy file. |
| **`p`** | **Paste** | Paste file. |

## üñ•Ô∏è Window & Navigation
Jangan sentuh mouse! Gunakan keyboard.

| Shortcut | Aksi | Keterangan |
| :--- | :--- | :--- |
| **`Spasi + v`** | **Vertical Split** | Belah layar Kiri-Kanan. |
| **`Spasi + h`** | **Horizontal Split** | Belah layar Atas-Bawah. |
| **`Ctrl + h`** | Focus Left | Pindah ke jendela Kiri. |
| **`Ctrl + l`** | Focus Right | Pindah ke jendela Kanan. |
| **`Ctrl + j`** | Focus Down | Pindah ke jendela Bawah. |
| **`Ctrl + k`** | Focus Up | Pindah ke jendela Atas. |
| **`Spasi + x`** | **Close** | Tutup file/buffer yang aktif. |

## üîç Search (Telescope)
Cari file dan kodingan secepat kilat.

| Shortcut | Aksi | Keterangan |
| :--- | :--- | :--- |
| **`Spasi + f + f`** | **Find File** | Cari file berdasarkan nama. |
| **`Spasi + f + w`** | **Find Word** | Cari teks/kata di DALAM seluruh file. |
| **`Spasi + f + o`** | **Find Old** | Cari file yang barusan dibuka (History). |
| **`Spasi + t + h`** | **Themes** | Ganti tema NvChad. |

## üíª Terminal & Coding Tools
Senjata utama SysAdmin.

| Shortcut | Aksi | Keterangan |
| :--- | :--- | :--- |
| **`Alt + i`** | **Float Term** | Terminal melayang (Cepat & Praktis). |
| **`Alt + h`** | **Horiz Term** | Terminal di bawah layar. |
| **`Spasi + f + m`** | **Format** | Rapikan kodingan otomatis (Beautify). |
| **`Spasi + /`** | **Comment** | Ubah baris jadi komentar (`#`). |
| **`Shift + k`** | **Hover Doc** | Lihat dokumentasi/penjelasan kode. |
| **`g + d`** | **Definition** | Lompat ke asal variabel/fungsi dibuat. |
